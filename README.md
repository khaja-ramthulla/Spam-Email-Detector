# Spam Email Classifier

A machine learning-powered web application that classifies emails and messages as **Spam** or **Not Spam (Ham)** using Natural Language Processing (NLP) techniques.

## Features

* Spam Detection
* Real-time Prediction
* Text Preprocessing
* Confidence-based Classification
* Clean and Responsive Web Interface

## Technologies Used

* Python
* Flask
* Scikit-Learn
* TF-IDF Vectorization
* Naive Bayes / Logistic Regression
* HTML, CSS, JavaScript

## How It Works

The application converts email text into numerical features using TF-IDF vectorization and uses a trained machine learning model to predict whether the message is spam or legitimate.
